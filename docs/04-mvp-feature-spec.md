## 0. MVP 목표

**커플이 어려운 대화를 Pause 안에서 시작하고, 텍스트와 음성 모두에서 비강제적 중립 개입이 실제로 대화를 늦추거나 완화하는지 검증한다.**

## 2. P0 상세

### 2.1 파트너 페어링

- 앱 최초 실행 시 초대 링크 생성 → B가 링크를 열고 연결 동의
- 완료 후 지속적인 커플 대화 공간을 생성
- 이후 매 대화마다 재페어링하지 않음
- 링크 만료·이미 사용된 링크·연결 취소 상태를 안내

### 2.2 대화 공간과 텍스트

- 대화방에는 메시지 목록과 텍스트 입력창이 항상 존재한다.
- 작성자가 전송을 누르기 전 초안을 분석한다.
- 위험도가 높은 표현이 감지되면 작성자에게만 개인 코칭 카드를 표시한다.
- 작성자는 원문 전송, 직접 수정, 제안 사용을 선택할 수 있다.
- 상대는 원래 초안이나 개인 코칭 내용을 볼 수 없다.
- 대화가 빠르게 격화되면 양쪽에 공유 넛지를 표시한다.
- 메시지 전송은 절대 강제로 차단하지 않는다.

### 2.3 Pause 내부 음성통화

- 대화방에서 통화 버튼을 누르면 상대에게 요청을 보낸다.
- 상대는 수락·거절할 수 있고, 요청 중·연결 중·연결됨 상태를 양쪽에 표시한다.
- 수락 후 양쪽이 동의하고 마이크 권한을 허용하면 통화를 시작한다.
- 통화 중에는 공유 넛지와 개인 코칭을 비강제로 표시한다.
- 누구든 종료할 수 있으며 종료 즉시 마이크와 분석을 중단한다.
- 네트워크 단절 시 재연결 안내를 표시하고, 분석 중단 상태를 명확히 알린다.

### 2.4 분석·개입

- 음향: 개인 기준 대비 음량, 발화 속도, 동시 발화·끼어들기
- 의미: 비난, 경멸, 방어, 담쌓기, 관계 위협 표현
- 텍스트: 관계 위협, 모욕, 반복 비난, 연속 전송, 과도한 대문자·구두점, 반복 요구
- 공유 넛지: 양쪽에 동일한 중립 문구
- 개인 코칭: 작성자·발화자에게만 표시되는 재작성·대기·요점 정리 제안
- 자동 발화, 자동 전송, 자동 통화 종료, 책임·공격성 점수는 금지

### 2.5 안전·프라이버시

- 분석 전 양쪽의 명시적 동의 필수
- 원본 음성 영구 저장 금지
- 개인 초안·개인 코칭 비공개
- 최소한의 신고·차단·안전 안내 제공
- 가정폭력·강압·즉각적 위험 신호가 감지되면 일반 중재를 멈추고 전문기관 안내로 전환하는 정책 필요

## 3. 명시적 비목표

범용 메신저, 그룹 채팅·통화, 영상통화, 파일·스티커, 데일리 퀴즈, 저널, 관계 점수·게임화, 책임 판정, 감시, 상대 동의 없는 녹음, AI 연인·상담사, 커뮤니티, 생체 데이터.

## 4. P1로 미루는 이유

통화 요약과 감정 그래프는 결과가 다음 싸움의 증거로 악용되지 않는지 검증한 뒤 개발한다. MVP는 실시간 개입과 양쪽 참여라는 핵심 가설에 집중한다.

## 5. 핵심 완료 기준

- 페어링된 커플이 대화방에서 텍스트를 주고받을 수 있다.
- 통화 요청·수락 후 Pause 내부 음성통화가 연결된다.
- 양쪽 동의 없이 통화 분석이 시작되지 않는다.
- 위험 초안에는 작성자에게만 코칭이 표시된다.
- 격화 신호에는 양쪽에 동일한 공유 넛지가 표시된다.
- 제안은 무시하거나 닫을 수 있다.
- 통화 종료·재연결·안전 상태가 명확히 표시된다.
- 원본 음성이 영구 저장되지 않는다.
